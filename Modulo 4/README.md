# Mis Ejercicios del Módulo 4

Este repositorio es mi bitácora de aprendizaje del **Módulo 4** del Bootcamp. Aquí fui subiendo los ejercicios de cada lección, pasando por objetos, clases, manipulación del DOM, lógica asíncrona y consumo de APIs.

Qué hice en cada etapa:

## Resumen de las Lecciones

### Lección 1: Objetos y Clases 
Aquí aprendí a organizar la información usando la Programación Orientada a Objetos (POO).
- **Objetos:** Creé un objeto `auto` (Toyota Corolla) para entender las propiedades.
- **Prototipos:** Usé la forma antigua de crear "moldes" para personas.
- **Clases y Herencia:** Creé una clase `Mamifero` y luego hice que `Perro` y `Gato` heredaran de ella para emitir sus propios sonidos ("Guau" y "Miau").
- **Arrays:** Jugué con una lista de estudiantes para sacar datos específicos.

### Lección 2: JavaScript Moderno (ES6) 
Aquí actualicé mi forma de escribir código a los estándares nuevos:
- **Variables:** Diferencia entre `var`, `let` y `const`.
- **Funciones Flecha:** Convertí funciones normales a funciones flecha `=>`.
- **Trucos nuevos:** Usé *Template Strings* (las comillas invertidas `` ` ``) y *Destructuring* para sacar datos de objetos más rápido.
- **Promesas:** Hice un ejercicio con `async/await` para simular una espera de 2 segundos.

### Lección 3: Manipulando el DOM (Interactividad) 
Aquí conecté el HTML con el JavaScript para que la página hiciera cosas reales:
- **Repetidor:** Un input que repite una palabra la cantidad de veces que le digas.
- **Cambiador de Color:** Una función para cambiar el fondo de un texto según el color que elijas.
- **Calculadora:** Hice una mini-calculadora que suma, resta, multiplica y divide dos números al mismo tiempo.
- **Inversor:** Escribes "Holanda" y te devuelve "adnaloH".

### Lección 4: Callbacks (Lógica asíncrona) 
La última lección, donde trabajé con funciones que reciben otras funciones:
- **Validar:** Pide un número y te avisa si es válido o no.
- **Temporizador:** Una suma que te hace esperar 5 segundos intencionalmente antes de dar el resultado.
- **Sumas Sucesivas:** Un cálculo que va sumando acumulativamente y decide si darte un mensaje de éxito o de error según el tamaño del número.

### Lección 5: Consumo de APIs (Fetch)
Conecté la aplicación con datos reales de internet usando la API de Studio Ghibli.
- **Fetch API:** Usé `fetch` para obtener datos externos (JSON).
- **Async/Await:** Manejé la asincronía para esperar la respuesta del servidor.
- **Optimización:** Implementé una lógica para guardar los datos localmente tras la primera carga.
- **Consola:** Visualización de datos estructurados directamente en la consola del navegador.

### Proyecto Final: Mini E-commerce (Cierre del Módulo)
El trabajo final donde integré todo lo aprendido (HTML + CSS + JS + DOM + APIs):
- **Catálogo Real:** Consumí la API de `FakeStoreAPI` para mostrar productos reales.
- **Renderizado Dinámico:** Usé `map` para dibujar las tarjetas de productos en el HTML.
- **Buscador en Vivo:** Implementé un filtro que busca productos mientras escribes.
- **Carrito:** Lógica para contar cuántos elementos agregas al carro.
- **Conversión de Moneda:** Transformé los precios de dólares a Pesos Chilenos (CLP) con formato de miles.
- **Estilos:** Usé CSS Grid y un encabezado *sticky* (pegajoso) para mejorar la navegación.
